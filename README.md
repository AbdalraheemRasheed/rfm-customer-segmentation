# RFM Customer Segmentation
Data analysis project focused on customer segmentation using RFM (Recency, Frequency, Monetary) model to uncover insights, identify VIP customers, improve retention, and support business decision-making.

## 📊 Key Insights
- VIP customers generate a large portion of total revenue.
- A high number of customers are one-time buyers, indicating low retention.
- At-risk customers have not purchased recently and need re-engagement.
- Revenue is concentrated among top customers.
- Certain regions outperform others in revenue contribution.
  
## 🛠️ Tools Used
* Python (Pandas, Matplotlib)
* MySQL
* Excel

## 📊 Analysis Includes
* Data cleaning & preparation
* RFM score calculation and segmentation
* Customer group profiling
* SQL queries for data exploration
* Pivot tables and charts in Excel
* Matplotlib visualizations

## 🚀 How to Run
1. Clone the repository
2. Install requirements: `pip install pandas matplotlib`
3. Open `analysis.py` in VS Code to view the Python analysis
4. Import `rfm-customer_analysis-SQL.sql` into MySQL to explore the data
5. Open `rfm-customer_analysis-excel.xlsx` to explore the pivot tables and charts
